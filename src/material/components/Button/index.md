## Example

```jsx
<div>
  <Button raised>
    Default
  </Button>
  <Button raised color="primary">
    Primary
  </Button>
  <Button raised color="accent">
    Accent
  </Button>
  <Button raised color="contrast">
    Contrast
  </Button>
  <Button raised color="accent" disabled>
    Disabled
  </Button>
  <input accept="jpg,jpeg,JPG,JPEG" id="file" multiple type="file" />
  <label htmlFor="file">
    <Button raised component="span">
      Upload
    </Button>
  </label>
  <Button raised dense>
    Dense
  </Button>
</div>
```
